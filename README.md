## ✦ About Usul Learning

<table>
<tr>
<td width="60%">

> *"And say: My Lord, increase me in knowledge."*
> — Qur'ān 20:114

**Usul Learning** is an Islamic education platform built on the conviction that authentic knowledge deserves a structured home.

The mission: transform scattered, fragmented Islamic learning into **organized, accessible, and enduring educational content** — rooted in the Qur'ān, grounded in the authentic Sunnah, and articulated through the Shāfi'ī tradition.

This is not a content feed. It is a **knowledge architecture**.

</td>
<td width="40%" align="center">

```
    ◈  METHODOLOGY  ◈

    Source  →  Qur'ān & Sunnah
    School  →  Shāfi'ī Madhhab
    Method  →  Structured Volumes
    Goal    →  Lasting Clarity
```

</td>
</tr>
</table>
