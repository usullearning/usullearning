### 🌿 About Usul Learning

**Usul Learning** is a structured Islamic education initiative built on the  
Qur'an and authentic Sunnah within the **Shafi'i tradition**.

We develop bilingual **(Arabic–English)** learning materials organized into  
carefully structured volumes — designed for clarity, authenticity, and long-term benefit.
